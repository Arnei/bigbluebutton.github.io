BigBlueButton is an open source web conferencing system for online learning.

The goal of the project is to enable remote students to have a high-quality online learning experience.  A key component of achieving this goal is giving the tools to engage remote students.   

This document gives you an overview of how to setup a development environment for BigBlueButton 2.0-beta (referred hereafter as BigBlueButton 2.0).


# Before you begin

You first need to setup a BigBlueButton 2.0 server.  See the instructions at [Install BigBlueButton 2.0](/2.0/20install.html).
